addonpy 

"addonpy" is a python module to provide addon / plug-in like functionality. This is simple addon interface provider.

Installation

pip install addonpy 

or download zip file and add unzipped location in PYTHONPATH variable so that python can find the module

Works On

Python: 2.7, 3.2, 3.3 & 3.4

Build CI: https://travis-ci.org/ninadmhatre/addonpy 

Current Build Status: 

Features

  1. Allows config file to set default values (addon-loader.info) [Please read wiki for more information]
  2. Allows you to specify which functions must addon have in order to consider it a valid one!
  3. Allows lazy_load mode, addon will be loaded when you try to get_instane of that module
  4. Allows nested addon calling (lazy_load mode break this functionalty) 
  5. Supports recursive directory search for addons, just specify top level directory
  6. Allows you to specify your own logger (using decorator) 
  7. Works on Python 2.7 & Python 3.* (Check TravisCI for more info)

Modules

  • addonpy: Main module which allows loading of addons and requires every addon to have .info file for lazy_load

  • addonpyLite: Same as above, but doesnt require .info file. Info file information is embedded into addon itself. No lazy_load mode

  • addonpyHelpers: Helper module

  • addonpyExecutor: Execute addons by order with try..except (try..catch) block arround addon being executed

  • IAddonInfo: Every addon should inherit from this class; used to retrieve addon related information

  • ILogger: Use if you are using your own logger and want addonpy to use the same. decorator around your logger.
